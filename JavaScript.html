1.如果函数没有被自身的对象调用时，this会变成全局对象。在web浏览器中，全局对象是浏览器窗口window。
function myFunction(){
    return this;
}
myFunction();  //返回window对象

2.将函数定义为对象的方法时，this会变成对象本身，因为对象拥有这个函数。
var myObject = {
    fname:"Dan";
    lname:"John";
    fullName:function(){
        return this.fname+" "+this.lname;
    }
}
myObject();   //返回this作为myObject的属性fname+lname

3.使用构造函数调用函数，构造函数本身中的this没有任何的值，this的值会在调用时new object时创建
function myFunction(arg1,arg2){
    this.attribute1=arg1;
    this.attribute2=arg2;
};
var x= new myFunction(a,b);  //函数中的this指代在对象创建的瞬间具有意义。

4.使用函数方法调用函数，call()和apply()是预定义的函数方法，调用时this指向第一个参数
function myFunction(arg1,arg2){
    return arg1+arg2;
};
myArray=[a,b];
myFunction.call(this,a,b); //call()方法第二个为参数数值
myFunction.apply(this,myArray); //apply()方法第二个为参数数组
