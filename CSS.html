str.split([separator[, limit]]);
以分隔符为准，移除分隔符并且返回一个数组，分隔符省略则返回数组包含一个由整个字符串组成的元素。
如果分隔符为空字符串，则把str转换为字符数组。
如果分隔符出现在字符串的开始或结尾，或两者都分开，分别以空字符串开头，结尾或两者开始和结束。因此，如果字符串仅由一个分隔符实例组成，则该数组由两个空字符串组成
'abcdef'.split('a'); //=>["", "bcdef"]
'abcdef'.split('f'); //=>["abcde", ""]
'abcdef'.split('abcdef');//=>["", ""]

如果正则表达式 separator 包含捕获括号（capturing parentheses），则其匹配结果将会包含在返回的数组中。
var myString = "Hello 1 word. Sentence number 2.";
var splits = myString.split(/(\d)/);
